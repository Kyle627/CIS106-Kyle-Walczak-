## This program takes iteration input from user and outputs a multiplication table with x(input) number of expressions 

Main Program
  Get iteration value
  Get Value
  Demonstrate for loop

Get iteration:
	Parameters:
	    None
	Process: 
	    Display prompt
	    Get iteration
	Return Value:
	    Iteration
Get Value:
	Parameters: 
	    None
	Process:
	    Display Prompt
	    Get Value
	Return Value:
	    Value

Demonstrate for loops(Iteration, value)
	Parameters:
	    Start 
	    Stop 
	    Increment 
	Process: 
	    Initialize count (0)
	    Loop for count from start to stop
	    Output line
	    Display count
	Return Value: 
	     None
