## This program takes test scores (amount of tests is dependent on user input) and averages the scores. 

Main Program
  Get Test Count
  Get Total
  Get Average
  Display Result

Get Test Count:
	Parameters:
	    None
	Process: 
	    Display prompt
	    Get test count
	Return Value:
	    count

Get Total:
	Parameters: 
	    Count
	Process:
		Initialize total
		Initialize counter
		Loop while counter < count
		    Display Prompt
			Get score
			Add score to total
			Increment counter
	Return Value:
	    Total

Get Average:

	    Count

	Return Value:
