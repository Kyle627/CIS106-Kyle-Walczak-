# This program takes inputted test scores from 0-100 and sorts from highest to lowest 

Main Program
    Get test scores
    Sort 
    Display result

Get test scores:
	Parameters:
	    None.
	Process:
	    Initialize array with [] null values 
	    Initialize while loop
	    Start loop, loop while input is true 
	        Display prompt 
	        Get test score
	        Validate input (negative ends loop and passes the array to sort array function)
	        
	Return Value: 
	    Test scores array


Sort array: 
	Parameters:
	    Test scores array
	Process:
	    Sort
	Return Value: 
	    Sorted array 


Display result: 
	Parameters:
	    Sorted Array
	Process:
	     Display array 
	Return Value: 
	    None
