# This program takes inputted test scores and returns min, max and average of entered scores. 

Main Program
    Get test scores
    Calculate min
    Calculate max
    Calculate avg
    Display result

Get test scores:
	Parameters:
	    None.
	Process:
	    Initialize array with [] null values 
	    Initialize while loop
	    Start loop, loop while input is true 
	        Display prompt 
	        Get test score
	        Validate input (negative ends loop and passes the array to sort array function)	        
	Return Value: 
	    Test scores array

Calculate min:
	Parameters:
	    Test scores array(sorted)
	Process:
	    Sort
	    Min = [0]
	Return Value: 
	    Min 

Calculate max:
	Parameters:
	    Test scores array(sorted)
	    count
	Process
	    Max = [Count]
	Return Val
	    max

Calculate avg: 
	Parameters:
	    Test scores array, count
	Process:
	  Average = Sum array divided by count
	Return Value: 
	    average

Display result: 
	Parameters:
	    Min, max, average
	Process:
	     Display result
	Return Value: 
	    None
